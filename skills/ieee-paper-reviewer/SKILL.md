# IEEE Paper Reviewer

This is a scaffold skill for IEEE-style manuscript review.

## Target behavior

Review technical papers using IEEE-oriented criteria:

- Novelty and significance
- Technical correctness
- Problem formulation and assumptions
- Methodological rigor
- Experimental design
- Baseline fairness
- Ablation completeness
- Complexity and scalability
- Dataset and metric validity
- Reproducibility and artifact availability
- Threats to validity and limitations
- IEEE venue fit and clarity

## Output direction

Future implementation should produce:

- Summary of contributions
- Major strengths
- Major weaknesses
- Required revisions
- Optional improvements
- Decision recommendation
- Author response strategy
